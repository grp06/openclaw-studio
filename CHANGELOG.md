# Changelog

## v0.1.0 — 2026-02-28

First official release of OpenClaw Studio.

### Highlights

- **Agent Canvas** — Visual agent management with drag-and-drop tiles, agent inspection panels, and real-time status indicators
- **WebChat** — Built-in chat interface with session management, message queuing, and streaming assistant responses
- **Gateway Connection** — Reliable WebSocket connection to OpenClaw Gateway with automatic reconnect, lock handling, and IPv4/IPv6 loopback support
- **Skills System** — Per-agent skills configuration, eligibility indicators, and safe deletion flow
- **Proxy Auth** — Browser auth passthrough detection and Studio access token enforcement for public binds
- **Reliability** — Config retry rebuilds from fresh snapshots, stale history drift handling, and assistant thinking stream leak prevention
